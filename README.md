# structure

> **Active development — not a finished product.**

Strength training app for building long-term training plans and tracking workouts.

## Running the app

_Linux only for now._

```sh
./scripts/run_linux.sh
```

Builds the Rust backend and launches the Flutter app on Linux.

## Tests

### Rust

```sh
cd backend && cargo test
```

### Flutter

```sh
cd frontend && flutter test
```
